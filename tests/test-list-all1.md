### 九、边界与异常场景
| 编号 | 功能项 | 测试方法 | 期望结果 |
|:---:|---|---|---|
| 9.1 | 无 API Key 启动 | 删除 LLM provider 的 API Key | 启动失败并报错"Missing API key" |
| 9.2 | 渠道未连接时发消息 | 个人微信未扫码时发送消息 | 提示未登录或忽略，不崩溃 |
| 9.3 | LLM 超时/故障 | 模拟 LLM 接口超时或返回 5xx | 自动 failover，或返回友好错误提示 |
| 9.4 | 超大图片上传 | 发送 >10MB 的图片 | 提示图片过大，服务不崩溃 |
