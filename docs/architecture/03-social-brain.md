## NEAT 神经进化

### 算法流程

```
1. 初始化种群
   └─> 150 个最小网络

2. 评估适应度
   └─> 在环境中测试

3. 物种分组
   └─> 基于拓扑相似度

4. 选择+交叉+变异
   ├─> 添加节点 (5%)
   ├─> 添加连接 (10%)
   └─> 权重变异 (80%)

5. 生成新一代
   └─> 重复直到收敛
```

### 应用场景

- 交易策略优化
- 对话策略学习
- 资源分配决策

## PAD 情绪模型

### 三维情绪空间

```
P (Pleasure): -1.0 (痛苦) ~ 1.0 (愉悦)
A (Arousal):  -1.0 (平静) ~ 1.0 (兴奋)
D (Dominance):-1.0 (顺从) ~ 1.0 (主导)
```

### 情绪状态示例

| 情绪 | P | A | D |
|------|---|---|---|
| 快乐 | 0.8 | 0.6 | 0.4 |
| 愤怒 | -0.6 | 0.8 | 0.7 |
| 悲伤 | -0.8 | -0.4 | -0.5 |
| 平静 | 0.3 | -0.5 | 0.2 |

## OCEAN 人格模型

### 大五人格维度

| 维度 | 高得分 | 低得分 |
|------|--------|--------|
| 开放性 (O) | 好奇、创新 | 传统、保守 |
| 尽责性 (C) | 自律、可靠 | 随意、冲动 |
| 外向性 (E) | 活跃、社交 | 安静、内向 |
| 亲和性 (A) | 合作、信任 | 竞争、怀疑 |
| 神经质 (N) | 敏感、情绪化 | 稳定、冷静 |

### 人格配置示例

```yaml
personality:
  ocean:
    openness: 0.8        # 创新型
    conscientiousness: 0.9  # 严谨型
    extraversion: 0.3     # 内向型
    agreeableness: 0.7    # 合作型
    neuroticism: 0.2      # 稳定型
```

## 记忆系统

### 五层记忆架构

| 层级 | 名称 | 容量 | 保持时间 |
|------|------|------|---------|
| STM | 短期记忆 | 7 轮对话 | 分钟 |
| LTM | 长期记忆 | 无限制 | 永久 |
| EM | 情景记忆 | 关键事件 | 永久 |
| PM | 程序记忆 | 技能 | 永久 |
| SM | 语义记忆 | 知识 | 永久 |
