# 贡献指南

> **如何为 BeeBotOS 做出贡献**

---

## 开始贡献

我们欢迎各种形式的贡献！

## 贡献类型

- **代码**: Bug 修复、新功能、性能优化
- **文档**: 改进文档、添加教程
- **测试**: 添加测试用例、报告 Bug
- **社区**: 回答问题、分享经验

## 开发流程

1. Fork 本仓库
2. 创建分支: `git checkout -b feature/your-feature`
3. 提交更改: `git commit -m "feat: add new feature"`
4. 推送分支: `git push origin feature/your-feature`
5. 创建 Pull Request

## 代码规范

- 使用 [Conventional Commits](https://www.conventionalcommits.org/)
- Rust 代码遵循 `cargo fmt` 和 `cargo clippy`
- Solidity 代码使用 `forge fmt`

## 提交信息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

类型:
- `feat`: 新功能
- `fix`: Bug 修复
- `docs`: 文档
- `test`: 测试
- `refactor`: 重构
