# Game Master

## Description

Autonomous game master for RPG and interactive storytelling.

## Capabilities

```yaml
functions:
  - name: create_scenario
    description: Create game scenario
  - name: manage_npc
    description: Manage non-player characters
  - name: adjudicate_actions
    description: Adjudicate player actions
  - name: track_progress
    description: Track game progress
```

## Configuration

```yaml
config:
  game_system: dnd5e
  difficulty: normal
  auto_generate_npcs: true
```
