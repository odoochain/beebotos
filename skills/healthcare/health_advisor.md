# Skill: Health Advisor

## Description

Autonomous health and wellness advisory agent.

```yaml
functions:
  - name: analyze_symptoms
    description: Analyze health symptoms
  - name: recommend_treatment
    description: Recommend treatment options
  - name: track_vitals
    description: Track vital statistics
  - name: schedule_reminders
    description: Schedule medication reminders
```
