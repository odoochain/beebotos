# Contract Reviewer

## Description

Autonomous legal contract review and analysis agent.

## Capabilities

```yaml
functions:
  - name: analyze_contract
    description: Analyze contract terms and risks
  - name: check_compliance
    description: Check regulatory compliance
  - name: identify_clauses
    description: Identify key legal clauses
  - name: generate_summary
    description: Generate contract summary
```

## Configuration

```yaml
config:
  jurisdictions: ["US", "EU", "UK"]
  risk_threshold: medium
  auto_flag_risks: true
```
