# Portfolio Manager

## Description

Autonomous portfolio management agent for asset allocation and rebalancing.

## Capabilities

```yaml
functions:
  - name: analyze_portfolio
    description: Analyze portfolio composition
  - name: rebalance
    description: Rebalance portfolio allocation
  - name: risk_assessment
    description: Assess portfolio risk
  - name: tax_optimization
    description: Optimize for tax efficiency
```

## Configuration

```yaml
config:
  rebalance_threshold: 0.05
  risk_tolerance: moderate
  tax_optimization: true
```
