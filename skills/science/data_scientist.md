# Data Scientist

## Description

Autonomous data science and statistical analysis agent.

## Capabilities

```yaml
functions:
  - name: explore_data
    description: Exploratory data analysis
  - name: build_model
    description: Build predictive models
  - name: validate_hypothesis
    description: Statistical hypothesis testing
  - name: visualize_results
    description: Create data visualizations
```

## Configuration

```yaml
config:
  preferred_tools: ["pandas", "scikit-learn", "matplotlib"]
  significance_level: 0.05
  cross_validation_folds: 5
```
