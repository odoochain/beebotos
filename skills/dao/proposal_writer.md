# Proposal Writer Skill

## Overview

Crafts well-structured DAO governance proposals.

## Capabilities

- Proposal drafting
- Specification writing
- Impact analysis
- Implementation planning
- Community communication

## Configuration

```yaml
name: proposal_writer
version: 1.0.0
category: dao
```

## Proposal Template

```markdown
# [Proposal Title]

## Summary
One-paragraph summary of the proposal.

## Motivation
Why is this proposal needed?

## Specification
Technical and implementation details.

## Financial Impact
Budget requirements and treasury impact.

## Timeline
Implementation schedule with milestones.

## Risks and Mitigations
Potential risks and how to address them.

## Success Metrics
How to measure the success of this proposal.

## Conclusion
Final summary and call to action.
```

## Example

Input: "Create a proposal for a bug bounty program with 500K BEE budget"

Output: Full proposal document following the template.
